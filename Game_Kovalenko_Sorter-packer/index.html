📦 Сортировщик-упаковщик

Ваша задача - сортировать и складывать предметы по их признаку!

📋 Как играть:

  • Клик ЛКМ по ящику → поместить товар
  • Цифры 1-5 на клавиатуре → сдать полный ящик
  • На 2 уровне: клик ПКМ по коробке чтобы открыть ее
  • На 3 уровне: наводите мышь на предметы для определения "экспресс" товаров, которые складываются отдельно
